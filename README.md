FarmingGame (Udemy)
Un jeu développé sous Unity avec C# dans le cadre d'un cours Udemy.

Description
C'est un jeu de ferme basique où l'on fait pousser des plantations, qui poussent de façon aléatoire. Pour que les plantations poussent, il faut passer la nuit ou atteindre 2h du matin.

Installation
Téléchargez le fichier ZIP puis décompressez-le. Un fichier .exe sera disponible pour lancer le jeu.

Utilisation
Pour jouer, une fois dans le jeu, il faut retourner le sol avec la houe. Ensuite, sélectionnez les graines souhaitées dans l'inventaire (touche I). Avant de planter, assurez-vous que le sol soit humide !
Pour vendre ou acheter de nouvelles plantations, accédez au coffre bleu situé à droite de la maison en appuyant sur la touche E près de celui-ci.
Pour passer la nuit et faire grandir vos plantations, entrez dans la maison, puis, une fois près du lit, appuyez sur la touche Espace, E ou cliquez avec le bouton gauche pour dormir.
Attention, pour que les plantes aient une chance de pousser, n'oubliez pas de les arroser chaque nuit !

Remerciements
Cours Udemy utilisé : Game development made easy. Learn C# using Unity and create your own crop growing game! par James Doyle.
